# OpenHobbyOS Documentation

OpenHobbyOS is a 32-bit x86 hobby operating system written from scratch in C and Assembly. It features a monolithic kernel with preemptive multitasking, a Linux-compatible syscall ABI, a framebuffer compositor, and support for running Doom, GTK4, Qt6, and other ported software.

This documentation covers the design, internals, and usage of the operating system.
